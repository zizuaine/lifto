# Lifto 🚀

A full-stack subscription platform combining gaming, charity, and rewards.

## Tech Stack
- React (Vite)
- Node.js + Express
- MongoDB
- JWT Auth

## Features
- Subscription system
- Score tracking
- Custom draw engine
- Charity integration

## Setup

### Client
cd Client
npm install
npm run dev

### Server
cd Server
npm install
node server.js
